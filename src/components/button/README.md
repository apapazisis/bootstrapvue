<h1>Button</h1>

<h2>Button tags</h2>

Default tag is Button

```html
<bv-button
    variant="success"
    size="lg"
    active
    disabled>Hallo</bv-button>
```

```html
<bv-button
    type="link"
    variant="success"
    size="lg"
    active
    disabled>Link</bv-button>
```

```html
<bv-button
    type="reset"
    variant="success"
    size="lg"
    value="Input Button Reset"
    active
    disabled></bv-button>
```

```html
  <bv-button
    type="submit"
    variant="success"
    size="lg"
    value="Input Button Submit"
    active
    disabled>Hallo</bv-button>
```

```html
<bv-button
    type="ibutton"
    variant="success"
    size="lg"
    value="Input Button"
    active
    disabled>Hallo</bv-button>
```
